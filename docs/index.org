* Usage

** Run a package

#+BEGIN_SRC fundamental
  $ nix run github:ShadowRZ/flakes#${FLAKE_PACKAGE?}
#+END_SRC

** As an overlay

*** In a flake

#+BEGIN_SRC nix
  {
    inputs = {
      nixpkgs.url = "github:NixOS/nixpkgs/nixos-unstable";
      shadowrz = {
	url = "github:ShadowRZ/flakes";
	inputs.nixpkgs.follows = "nixpkgs";
      };
    };

    outputs = { self, nixpkgs, shadowrz, ... }: {
      # Reference packages
    };
  }
#+END_SRC

You can also use =github:ShadowRZ/flakes?dir=pkgs= to only get packages.
